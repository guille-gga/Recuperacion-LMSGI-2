De texto o de seleccion:

- Texto:
```

<input type="text" id="campoContenido" placeholder="Mensaje..." required>

```
- Type: Tipo que se recoge, text en este caso.

- id: nombre que tu pongas al id

- laceholder: es como el mensaje qye se muestra en el campo cuando no has seleccionado para escribir.

- required: los campos deben estar completos.


- Selección:

```
<input type="checkbox" id="campo" name="contenido" required>
```
- Type: Tipo que se recoge, checkbox en este caso para selecciones.

- id: nombre que tu pongas al id

- name: nombre asignado

- required: los campos deben estar completos.
